- Git 설치가 어렵거나 실습 환경 구축이 번거로운 경우, [Codeonweb](https://codeonweb.com/#/)과 같은 온라인 Git 실습 환경을 이용할 수 있다.
- Codeonweb 웹사이트에 접속하여 회원가입 후 "실습" 메뉴에서 Git 환경을 선택하면 웹 브라우저 상에서 Git 명령어를 실행해 볼 수 있는 터미널이 제공된다.
